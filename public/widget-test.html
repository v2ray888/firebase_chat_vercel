小部件测试页面

说明

这是一个用于测试简化版小部件的页面。小部件应该会出现在页面的右下角。

注意：请确保开发服务器正在运行（npm run dev），并且 widget.js 文件存在于 public 目录中。

测试步骤

  • 检查页面右下角是否出现小部件图标
  • 点击小部件图标，确认聊天窗口是否正常打开
  • 在输入框中输入消息并发送，检查是否能收到回复
  • 点击聊天窗口右上角的关闭按钮，确认窗口是否正常关闭
  • 检查小部件的样式是否符合预期

故障排除

如果小部件没有显示，请按以下步骤检查：

  • 打开浏览器开发者工具（F12），查看控制台是否有错误信息
  • 确认 widget.js 文件是否正确加载（在 Network 标签中查看）
  • 检查 data-widget-id 属性是否正确设置
  • 确认开发服务器是否正常运行
