Detailed Widget Debug Test Page

This is a detailed test page for debugging the NeonSupport widget.

故障排除指南:

  1. 确保开发服务器正在运行 (npm run dev)
  2. 打开浏览器开发者工具 (F12) 查看控制台日志
  3. 检查网络选项卡中是否有404错误
  4. 确保widget ID在数据库中存在

已知有效的Widget IDs:

  • 57f13bb3-7819-45db-9961-bce85b83f3d2 (163网站)
  • 6f0937e0-3be2-4600-a711-8ac712c0e4da (霓虹示例网站)
  • 4a1f426e-3643-43eb-8d2f-96c5515ac743 (百度)

注意: 确保开发服务器在端口9002上运行，这些widget才能正常工作。

测试步骤:

  1. 检查widget容器是否存在于DOM中
  2. 验证widget脚本是否正确加载
  3. 检查浏览器控制台中的错误信息
  4. 确认开发服务器在端口9002上运行
  5. 验证widget ID是否存在于数据库中
